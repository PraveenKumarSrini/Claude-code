# Petrol and Diesel Prices 

## Tech Stack
- Maven, Java 21 , Spring boot 
- No external API keys needed — use mock pricing data data

## Code Style
- Follow Coding conventions and best practices in Java
- REST API Standards
- 4-space indentation (No tabs)

## Project Goal
Build a Petrol and Diesel Price dashboard API with:
1. GET /health — returns status and uptime
2. GET /petrol/:city — returns mock pricing data for 5 cities
3. GET /diesel/:city — returns mock pricing data for 5 cities
4. GET / — HTML dashboard showing Petrol Diesel Price for 5 cities
